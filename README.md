# My Cow

My Cow is a mobile application built with React Native and Expo to help manage a dairy farm. It allows users to track cattle, milk production, health records, and financial transactions.

## Features

*   **Cattle Management:** Add, view, and manage individual cow details.
*   **Milk Records:** Log and monitor milk production for each cow.
*   **Health Records:** Keep track of health events and treatments.
*   **Transaction Management:** Record income and expenses related to the farm.
*   **Event Tracking:** Log important events for each cow, such as insemination and calving.
*   **Internationalization:** Support for multiple languages.

## Getting Started

### Prerequisites

*   Node.js
*   Expo CLI
*   Android Studio or Xcode for device emulation

### Installation

1.  Clone the repository:
    ```sh
    git clone <your-repository-url>
    ```
2.  Install NPM packages:
    ```sh
    npm install
    ```

## Available Scripts

In the project directory, you can run:

*   `npm start`: Runs the app in development mode.
*   `npm run android`: Deploys the app to a connected Android device or emulator.
*   `npm run ios`: Deploys the app to the iOS simulator.
*   `npm run web`: Runs the app in a web browser.
*   `npm test`: Runs the test suite.
*   `npm run lint`: Lints the code.
*   `npm run build:android`: Builds the Android app for production.
