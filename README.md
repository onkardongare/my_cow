## Organic Farming Mobile Application

Welcome to the MyCow Mobile Application repository! This project delivers a comprehensive mobile solution for dairy farm operations, developed using React Native and styled with NativeWind, a utility-first styling library inspired by Tailwind CSS.


### Prerequisites
Before you begin, ensure you have the following installed:

Node.js

Expo CLI (for managing the React Native project)

Git (for version control)

### Installation
Clone the Repository:

```bash
git clone https://github.com/yourusername/MyCow.git
```

Navigate to the Project Directory:
```bash
cd MyCow
```

Install Dependencies:

```bash
npm install
```

### Set Up Environment Variables:

Create a .env file in the root directory and add the necessary environment variables, such as API endpoints and secret keys.

Styling with NativeWind
This project utilizes NativeWind for styling, bringing the utility-first approach of Tailwind CSS to React Native. NativeWind ensures a consistent styling experience across all platforms, improving developer experience and code maintainability.


### Start the Development Server:

```bash
npx expo start
```
This command will launch the Expo development server, providing options to run the app on an emulator, a physical device, or a web browser.

### Access the Application:

Use the Expo Go app on your mobile device or an emulator to scan the QR code displayed in the terminal or browser.

Project Structure
